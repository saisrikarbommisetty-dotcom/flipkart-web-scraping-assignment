# Flipkart Web Scraping Assignment

## Overview
This repository contains a Python-based web scraping assignment focused on extracting laptop listings from Flipkart search pages.

## Files Included
- Web_Scraping_Assignment.ipynb – Scraping logic and data handling
- flipkart_laptops_dataset.csv – Extracted dataset (primary)
- Web_Scraping_Presentation.pptx – Project presentation

## Tools Used
- Python
- Requests
- BeautifulSoup
- Pandas

## Notes
Some attributes may contain missing values due to dynamic website rendering and listing-page limitations.
